
#  Flutter ECommerce-Task
E-commerce mobile app that allows users to browse products, additems to a cart,
and place orders. 

## Application Architecture
Application is build with Clean Architecture of Robert C. Martin (Uncle Bob) 

## App Features
Browse Products and display them to user 
Search User Products By Catagories
Add To Favorite
Add To Cart
Offline Mode "when the internet is disconneted display from cached memory otherwise display from apis"
Check out
Allow users to register/login before placing an order
